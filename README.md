# Tortuga AI Skills

Public agent skills for Claude Code by [Tortuga AI](https://bytortuga.com).

## Install

```bash
npx skills add https://github.com/tortuga-ai/skills
```

## Skills

| Skill | Description |
|-------|-------------|
| `tortuga:context-cost` | Break down what's eating the Claude Code context window |
| `tortuga:update-claudemd` | Update CLAUDE.md from latest git commits or full codebase scan |
| `tortuga:image-check` | Scan codebase for oversized images and convert to WebP |
| `tortuga:submit-to-indexnow` | Submit URLs to IndexNow for immediate Bing indexing |
